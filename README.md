# cycling-trainer-simulator

Indoor Cycling Trainer Simulator
Cycling Trainer Simulator is a lightweight Python desktop application that connects directly to your Wahoo KICKR CORE smart trainer, giving you powerful and realistic training options beyond the standard apps.
Core Features:

Constant Power Mode: Set and maintain any target wattage. The app automatically controls the trainer to deliver steady resistance — perfect for structured training sessions, FTP tests, or sweet-spot workouts.
GPX Route Simulator: Upload any .gpx file and ride the real route indoors. The app reads elevation data from the GPX file and dynamically adjusts the KICKR CORE’s resistance so you feel every climb and descent. Altitude gain directly translates into increased power requirements, creating a highly realistic virtual ride experience.

Additional Highlights:

Real-time power, cadence, speed, and heart rate monitoring (when paired with compatible sensors).
Simple, clean interface focused on ease of use.
Built in Python using libraries like bleak (for Bluetooth LE communication with the KICKR).

Whether you're doing focused power training or want to pre-ride a challenging mountain route from your next gran fondo, WahooSim turns your KICKR CORE into a versatile and immersive smart trainer companion.
